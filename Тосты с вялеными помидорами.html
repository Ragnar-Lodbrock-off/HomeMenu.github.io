Ингредиенты
Продукт Количество
Хлеб по вкусу
Вяленые помидоры сколько есть
Творожный сыр по вкусу
Бальзамический соус по вкусу

Инструкция приготовления

  1. Хлеб поджарить на сковороде или в тостере.
  2. Смазать творожным сыром.
  3. Выложить помидоры из банки на творожный сыр.
  4. Украсить бальзамическим соусом зигзагом поперёк куска хлеба.

Примечание: можно украсить зеленью.

Вяленые помидоры

Ингредиенты для вяления помидоров
Продукт Количество
Помидоры черри по вкусу
Соль по вкусу
Растительное масло по объёму банок с помидорами
Прованские травы по вкусу
Чеснок по вкусу

Инструкция приготовления

  1. Нарезать помидоры на четвертинки, удалить ножки.
  2. Посолить.
  3. Вялить 6-8 часов в суховее до изюмного состояния.
  4. Разогреть масло не до кипения.
  5. Вялить 6-8 часов в суховее до изюмного состояния.
  6. Чеснок мелко порубить.
  7. Поместить в прокипячёную банку помидоры, сверху прованские травы и чеснок, повторить слои до конца банки.
  8. Залить горячим (не кипящим!) маслом.
  9. Закрыть банку (не закатывать!) настаивать 3-4 дня. После можно есть.
